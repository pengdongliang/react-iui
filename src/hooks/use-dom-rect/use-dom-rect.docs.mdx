### 功能

- 获取dom元素大小及其相对于视口的位置

### Props

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| target | 目标元素, 支持ref, dom, ()=>dom | `BasicTarget` | - |

### Result

| 参数 | 说明 | 类型 |
| --- | --- | --- |
| height | 高度 | `number` |


### 示例

```tsx
const [rect] = useDomRect(document.getElementById('test'))

return <div id="test">test</div>
```

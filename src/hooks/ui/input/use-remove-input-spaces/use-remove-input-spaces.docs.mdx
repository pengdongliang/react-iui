### 功能

- 去除input内容首尾空格, 支持失去焦点和回车

### Props

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| regex | 替换正则 | `RegExp` &#124; `false` | `/(^\s*)|(\s*$)/g` |
| formName | form表单name | `string` | `Input id` |
| pressEnterFlag | 回车是否去除空格 | `boolean` | `true` |
| onChange | onChange | `React.ChangeEventHandler` | - |


### 示例

```tsx
import {useRemoveInputSpaces} from './use-remove-input-spaces'
import {Form, Input} from 'antd'

const [form] = Form.useForm()
const removeInputSpacesEvent = useRemoveInputSpaces({})

return (
  <Form form={form}>
    <Form.Item name="input">
      <Input {...removeInputSpacesEvent} />
    </Form.Item>
  </Form>
)
```

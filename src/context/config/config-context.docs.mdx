### 功能

- 为组件提供统一的全局化配置

### Props

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| children | children | ReactNode | - |
| antdConfig | antd配置 | AntdConfigProviderProps | - |
| cssinjsConfig | cssinjs配置 | StyleProviderProps | - |
| theme | 主题配置 | BaseThemeProps | - |
| themeConfigMode | 主题类型, 默认为merge, merge=合并每一层, replace=替换, replaceAntd=仅替换antdTheme | 'merge' &#124; 'replace' &#124; 'replaceAntd' | 'merge' |
|  locale  | 国际化语言代码   | string   |  zh-CN   |
|  localeConfig  |  国际化配置  |  LocaleConfigType  |   -  |

### 示例

import { ConfigProvider } from './config-context-provider'

```tsx live
<ConfigProvider />
```
